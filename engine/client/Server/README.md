## NSquad 서버 실행 가이드

### 처음 실행 가이드

* NSquad 서버는 .NET 8.0 기반으로 동작하므로 처음 실행하시는 경우 .NET 8.0을 설치하셔야 합니다. [.NET 8.0 다운로드 페이지](https://dotnet.microsoft.com/ko-kr/download/dotnet/8.0)에서 SDK 8.0을 다운로드하여 설치하시기 바랍니다.
