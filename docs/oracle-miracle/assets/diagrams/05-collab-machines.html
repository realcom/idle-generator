IDLE Forge 협약기간 내 핵심 협업 머신 3종
Trigger AI · 맵 에디터 · 다중 모드 지원 — 본 사업의 신규 개발 핵심
M1
Trigger AI 협업 머신
입력
자연어 정의
스폰·사망·영역 진입·시간 조건 + NPC·몬스터 행동 패턴
출력
검증된 게임 행동 로직 + 데이터 (자동 변환)
활용 도메인
4X 연맹 NPC 외교·전쟁, 서바이벌 적 웨이브·보스, 하우징 NPC 동적 이벤트
M2
맵·레벨 에디터 AI 머신
입력
자연어 정의
지형·오브젝트 배치·이벤트 포인트
출력
맵·레벨 초안 → 사내 디자이너 후처리·확정 협업
활용 도메인
닌자 서바이벌 2 하우징 영지 템플릿, 서바이벌 스테이지 맵 대량 양산
M3
다중 모드 지원 패키지
입력
게임 모드 정의
서바이벌·하우징·4X·방치형 등
출력
모드별 고유 콘텐츠 타입 + 도메인 스킬 세트 (단일 플랫폼 제공)
활용 도메인
B2B 라이선시가 자신의 게임 모드에 맞게 워크플로우 선택·조합
통합 산출
검증 사례 게임 닌자 서바이벌 2 + 향후 B2B 라이선시 게임
세 협업 머신의 산출물을 사내 디자이너·기획자가 후처리·확정 → IDLE Forge 표준 컴포넌트로 패키지화 → 자체 게임 및 B2B 라이선시 게임에 동시 활용. 협약 종료 후 SDK·문서·표준 라이선스 계약 정리하여 '28년 외부 공급 시작.
3종 머신의 정식 개발이 협약기간 8개월의 핵심 시간 비용 영역
