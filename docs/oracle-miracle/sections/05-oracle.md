## 5-1. 오라클 제공 서비스와 아이템의 협업 가능성

### □ 협업의 기본 구도

당사의 창업아이템 IDLE Forge는 AI 콘텐츠 생성 도구, 자체 모바일 게임 엔진, 라이브 운영 파이프라인이 결합된 통합 플랫폼입니다. 본 사업에서는 이 세 영역에 필요한 클라우드·AI 워크로드를 OCI(Oracle Cloud Infrastructure) 서비스로 구축하며, 단순한 호스팅이 아닌 콘텐츠 생성에서 라이브 운영까지의 워크플로우 전반을 클라우드 위에서 운영합니다.

OCI는 2026년 GA된 Enterprise AI 스택을 통해 Generative AI Service, Generative AI Agents, Data Science를 통합 제공하고 있으며, 당사는 이 스택을 게임 스튜디오 운영에 최적화된 형태로 활용합니다.

### □ 활용 OCI 서비스 (네 영역)

  ○ 콘텐츠 생성 — OCI Enterprise AI 스택
  - **OCI Generative AI Service**: 닌자 서바이벌 2의 4X 영지·연맹 미션, 하우징 오브젝트, 서바이벌 적·웨이브 패턴 등 대규모 콘텐츠를 자동 생성합니다. Cohere Command A, Grok 4.1, OpenAI gpt-oss 등 다수 모델을 단일 API에서 운영할 수 있으며, Model Import 기능으로 당사가 파인튜닝한 게임 도메인 모델도 가져올 수 있습니다.
  - **OCI Generative AI Agents**: 새로 생성된 콘텐츠가 기존 자산과 밸런스·톤이 충돌하는지 검사하는 멀티스텝 에이전트를 운영합니다. RAG, 관측성, 감사 로그가 내장되어 있어 별도 인프라 없이 검증 파이프라인을 구축합니다.
  - **Oracle Database 23ai · AI Vector Search**: 라이브 게임 콘텐츠 임베딩을 저장하고, 신규 콘텐츠 생성 시 유사 사례를 RAG로 참조하여 닌자 IP 특유의 톤을 유지합니다. DB 내부 네이티브 기능이라 별도 벡터 DB 운영이 필요 없습니다.

  ○ 엔진 운영 — OCI 인프라
  - **OCI Compute + OKE (Kubernetes Engine)**: 4년간 검증된 자체 .NET 서버를 OCI Seoul/Chuncheon 리전에 배포합니다. Virtual Nodes를 활용해 인프라 관리 인력 없이 4X 연맹 전쟁 폭증 트래픽에 자동 대응합니다.
  - **OCI Object Storage + CDN**: AI가 양산한 신규 영지·이벤트 콘텐츠를 모바일 앱에 주 단위로 핫 패치합니다. 네이티브 OCI CDN이 글로벌 전송을 담당합니다.
  - **OCI APM, Logging Analytics, Data Science**: 라이브 운영의 분산 트레이싱, 이상 패턴 탐지, 결제 가능성 예측 모델을 운영합니다. APM 무료 티어(시간당 1,000건 트레이스)로 초기 운영 비용을 절감합니다.

  ○ 운영 자동화 — 다국어와 파이프라인
  - **OCI Language + Generative AI Agents**: 30개 언어 번역과 Custom Glossary로 다국어 CS를 자동 응대합니다. 닌자 캐릭터명·스킬명 같은 IP 고유 용어가 자동 일관 유지됩니다.
  - **OCI Functions (Serverless)**: 기획자가 콘텐츠 정의를 푸시하면 컴파일, 업로드, CDN 캐시 무효화까지 자동 수행하는 파이프라인을 운영합니다.

  ○ 인게임 AI — 동적 NPC와 이벤트
  - **OCI Generative AI Agents (RAG)**: 4X 연맹 시스템의 NPC가 플레이어의 점령 패턴을 학습해 외교·전쟁 결정을 동적으로 내립니다. 하우징 영지를 방문하는 NPC는 플레이어 진행도와 보유 자원에 맞춰 매번 다른 대화·퀘스트를 제공합니다.

### □ 리전 운용 계획

OCI Generative AI Agents 서비스의 가용 리전(아시아 권역에서 오사카가 한국 최근접)과, 게임 서버의 저지연 요구를 동시에 만족시키기 위해 다음과 같이 분리 운용합니다.

| 워크로드 | OCI 리전 | 사유 |
|---|---|---|
| 게임 서버 (실시간 트래픽) | Seoul · Chuncheon | 한·일 유저 저지연 |
| AI Agents (콘텐츠 생성·검증) | Osaka (Japan East) | Generative AI Agents 가용 최근접 리전 |
| 글로벌 CDN | 멀티 리전 + Edge | 패치 콘텐츠 글로벌 전송 |

리전 분리는 한국 PIPA, EU GDPR, 중국 PIPL 등 지역별 데이터 거버넌스 요건 대응에도 활용합니다.

## 5-2. 글로벌 기업 협업 프로그램 성공을 위한 핵심성과 및 전략

### □ 협업으로 창출할 핵심성과

  ○ 신규 IP 출시 주기 단축
  - 전통 인디 스튜디오에서 신작 1종 출시까지 1.5~2년이 소요됩니다. 당사는 IDLE Forge와 OCI Enterprise AI 스택을 결합해 동일 작업을 3개월 수준으로 단축하는 것을 목표로 합니다.
  - 협약기간 내 닌자 서바이벌 2의 베타 출시가 이 성과의 1차 실증입니다.

  ○ 5인 미만 조직으로 글로벌 라이브 운영
  - OCI Virtual Nodes로 인프라 관리 인력을, OCI Language·Agents로 다국어 CS 인력을 각각 최소화합니다.
  - 동일 매출 규모의 전통 게임사 30인 조직 대비, 당사는 합류 확정 4인과 협약기간 내 채용 2인 총 6인으로 운영합니다.

  ○ 닌자 서바이벌 2 월 매출 5억원 도달
  - 2027년 안정화 시점 DAU 80,000, 결제 유저 5%, ARPPU(인당 결제 평균) 미화 80달러를 가정합니다. 산식상 IAP(인앱결제) 약 4.48억원, IAA(인게임광고) 약 0.56억원으로 월 5억원에 도달합니다.
  - 동시기 연 매출 약 40억원을 예상하며, 4X 패치 이후 월 10억원 수준 성장이 가능합니다.

  ○ IDLE Forge B2B SaaS 외부 공급
  - 닌자 서바이벌 2로 플랫폼이 실증되면, 2028년부터 IDLE Forge를 다른 인디 게임사에 월 구독과 매출 분배 형태로 공급합니다.
  - 공급 채널의 하나로 OCI Marketplace 등록을 검토합니다.

### □ 성과 실현 전략

  ○ 협약기간 8개월을 OCI 네이티브 전환의 결정적 시기로 활용합니다. IDLE Forge 워크플로우 전체를 OCI 위로 이식하고, 협약 종료 시점에는 그 위에서 즉시 글로벌 라이브가 가능한 상태에 도달합니다.

  ○ 도입 리스크를 최소화합니다. 다년간 함께 글로벌 라이브 게임을 만들어 온 동일 팀이 그대로 합류하므로 팀 빌딩 리스크가 없고, 4년 검증된 자체 엔진 위에 OCI를 얹는 구조이므로 엔진 안정성 리스크도 없습니다. 새로 도입하는 요소는 OCI 한 가지로 좁혀 본 사업의 협업 가치에 집중합니다.

  ○ 퍼블리싱 파트너 확보는 협약 후반부의 주요 과제입니다. 당사는 현재 별도의 퍼블리셔를 확보하지 않은 상태이며, Oracle for Startups 프로그램과 Oracle AI World 2026 행사 참여를 한·중·동남아 퍼블리셔 컨택의 기회로 활용할 계획입니다.

  ○ 사례 공유와 시장 인지도 축적을 병행합니다. 컨퍼런스·블로그·기술 문서를 통해 IDLE Forge 운영 경험을 공개하여, 2028년 외부 공급 시점에 필요한 시장 신뢰를 미리 확보합니다.

## 5-3. 협업 로드맵

### □ 협약기간('26.04 ~ '26.11) 단계별 도입

협약기간 8개월을 두 달 단위 네 구간으로 나누어 OCI 워크로드를 단계적으로 도입합니다.

| 구간 | OCI 도입 작업 | 게임 마일스톤 |
|---|---|---|
| '26.04 ~ '26.05 | OCI 환경 셋업, OKE에 게임 서버 PoC, Object Storage·CDN 연동 | 데이터 이관 완료, 사내 OCI 학습 |
| '26.06 ~ '26.07 | Generative AI Service·Agents로 콘텐츠 양산 파이프라인 전환 | 서바이벌+하우징 코어 루프 MVP, 클로즈드 베타 |
| '26.08 ~ '26.09 | Vector Search RAG, Language CS 자동화, Functions 파이프라인 | 4X 시스템 구축, AI 리소스 양산 가속 |
| '26.10 ~ '26.11 | APM·Logging Analytics·Data Science 운영 KPI 대시보드, 인게임 NPC Agents | Oracle AI World 2026(라스베가스 10/26~29) 참여, 글로벌 베타 출시 |

### □ 협약 종료 이후 중기 계획

  ○ 2027년 — 정식 출시와 라이브 운영
  - '27년 1월 닌자 서바이벌 2 글로벌 정식 출시, OCI 위에서 라이브 운영을 지속합니다.
  - 신규 시즌·연맹 이벤트는 OCI Functions 기반 자동 배포 파이프라인으로 주 단위 업데이트합니다.
  - 매출은 4분기 안정화 시점에 월 5억원 도달을 목표로 합니다.

  ○ 2028년 — IDLE Forge B2B 공급 시작
  - IDLE Forge를 외부 인디 게임사에 SaaS 형태로 공급합니다. 1호 고객사 확보를 연중 목표로 두며, 공급 채널의 하나로 OCI Marketplace 등록을 진행합니다.
  - 동시에 멀티 타이틀 전략으로 IDLE Forge 기반 신규 IP 1~2종을 자체 출시합니다.

  ○ 2029년 이후 — 멀티 타이틀과 외부 공급 확대
  - 자체 출시 타이틀 누적 3종 이상, IDLE Forge 외부 고객사 다수 확보를 목표로 합니다.
  - Oracle 글로벌 커뮤니티에서 사례 공유를 이어가며, 당사의 시장 인지도를 안정화합니다.
