# 2. 실현가능성 (Solution)

## 2-1. 창업아이템 현황 (준비 정도)

### □ IDLE Forge 플랫폼 구성 자산 (현재 운영 중)

IDLE Forge는 다음 세 레이어로 구성된 통합 플랫폼입니다. 각 레이어의 핵심 자산이 본 사업 신청 시점에 이미 운영되고 있습니다.

  ○ 콘텐츠 양산 레이어 — AI 콘텐츠 자동 생성 하네스
  - 게임 콘텐츠(아이템·스킬·유닛·맵·이벤트 트리거 등)를 자연어 정의로 입력하면 게임 엔진이 로드 가능한 데이터로 자동 변환하는 워크플로우를 사내 도구로 운영하고 있습니다.
  - 도메인 특화 생성 스킬 30종 이상(gen-items, gen-skills, gen-units 등)이 구축되어 있으며, 스키마 검증과 밸런스 시뮬레이션을 자동 수행하는 검증 에이전트가 함께 동작합니다.
  - AI 아트 워크플로우(Midjourney, LayerAI 등 결합)로 인게임 아이콘·배경·UI·배너를 외주 없이 대량 생산하고 있습니다.

  ○ 엔진 레이어 — 자체 모바일 게임 엔진 idlez
  - 4년 이상 운영된 자체 모바일 게임 엔진으로, 누적 170만 다운로드의 라이브 게임 2종(닌자키우기·닌자 서바이벌)을 통해 안정성이 입증되었습니다.
  - 단일 코드 기반의 비동기 통신 처리, 글로벌 멀티 리전 배포, 콘텐츠 핫 패치 시스템을 지원합니다.

  ○ 라이브 운영 파이프라인 — 검증·배포·모니터링 자동화
  - 콘텐츠 컴파일·검증·배포 파이프라인을 사내 스크립트로 운영 중이며, 본 사업에서 OCI Functions와 연계하여 완전 자동화 단계로 끌어올립니다.
  - 현재 라이브 운영 중인 게임 2종의 실제 운영 데이터가 플랫폼 개선의 1차 피드백 원천으로 활용됩니다.

### □ 신청 시점의 단계별 현황

| 영역 | 현재 단계 |
|---|---|
| AI 콘텐츠 생성 하네스 | 사내 운영 중 (gen-* 스킬 30+종, 검증 에이전트 가동) |
| AI 아트 워크플로우 | 사내 운영 중 (라이브 게임 아트 외주 의존도 0) |
| idlez 엔진 | 4년 운영 검증, 라이브 게임 2종 가동 |
| OCI 네이티브 전환 | 협약 개시 후 본격 착수 ('26.04~) |
| 검증 사례 게임 (닌자 서바이벌 2) | 컨셉·코어 루프 설계 완료, MVP 제작 직전 |
| 라이브 운영 매출 | 월 약 3,000만원 (이관 게임 2종) |

본 사업에서 새로 도입하는 핵심 요소는 **OCI 네이티브 전환 한 가지**로 좁혀져 있습니다. 다른 자산은 모두 운영 중인 자산을 정식 플랫폼화하는 작업입니다.

### □ 실증 데이터로 확인한 플랫폼 가능성

  ○ 자체 도구를 활용한 라이브 게임 신규 모드 추가 실험 (2025년)
  - 운영 중 게임에 서바이벌 + 채집(광산) 모드를 사내 AI 도구로 단기간 추가하여 실증 테스트를 진행하였습니다.
  - 결과: 리텐션 +5%p, 수익 지표 +5%p
  - 이 데이터는 사내 도구가 단순 보조가 아닌 라이브 운영의 핵심 자산이 될 수 있음을 보여주며, 본 플랫폼을 정식 IDLE Forge로 확장할 때의 시장성을 1차 검증합니다.

  ○ 라이브 게임 2종의 안정적 매출 베이스
  - 월 약 3,000만원의 안정 매출은 플랫폼 개발 기간의 운영 자금이자, 플랫폼 개선을 위한 실시간 피드백 원천입니다.
  - 외부 게임을 통해 플랫폼을 검증해야 하는 일반 도구 개발사와 달리, 당사는 사내 라이브 게임으로 플랫폼을 실시간 검증할 수 있는 구조를 보유합니다.

## 2-2. 창업아이템 실현 및 구체화 방안

### □ IDLE Forge 플랫폼 완성 방안 (협약기간 내)

  ○ 콘텐츠 양산 레이어의 OCI Enterprise AI 전환
  - 현재 외부 LLM API 기반으로 운영 중인 콘텐츠 생성 워크플로우를 OCI Generative AI Service, Generative AI Agents, Database 23ai AI Vector Search로 전환합니다.
  - 데이터 주권과 비용 예측 가능성을 확보하며, Model Import 기능으로 당사가 파인튜닝한 게임 도메인 모델을 자산화합니다.

  ○ 엔진 레이어의 OCI 네이티브 운영
  - idlez 엔진을 OCI Compute과 OKE(Kubernetes Engine)로 멀티 리전 배포합니다. Virtual Nodes 기반 서버리스 운영으로 인프라 관리 인력 부담을 줄입니다.
  - Object Storage와 CDN으로 콘텐츠 핫 패치를 글로벌 자동 전송합니다.

  ○ 라이브 운영 파이프라인 완성
  - OCI Functions로 콘텐츠 컴파일·배포 파이프라인을 완전 자동화합니다. 기획자가 콘텐츠 정의를 푸시하면 빌드와 글로벌 배포까지 무인 진행됩니다.
  - OCI Language와 Generative AI Agents로 다국어 CS 자동화 모듈을 추가하여, 5인 미만 조직으로 글로벌 라이브 운영이 가능한 구조를 완성합니다.
  - OCI APM과 Logging Analytics로 운영 KPI 대시보드와 이상 패턴 탐지 모듈을 구축합니다.

  ○ 인게임 AI 모듈 추가
  - OCI Generative AI Agents 기반 동적 NPC와 동적 이벤트 모듈을 IDLE Forge의 표준 컴포넌트로 추가합니다. 향후 IDLE Forge로 제작되는 모든 게임이 이 모듈을 활용할 수 있도록 패키지화합니다.

### □ 검증 사례 게임 — 닌자 서바이벌 2

  ○ 검증 사례 게임의 역할
  - 닌자 서바이벌 2는 IDLE Forge로 제작하는 첫 검증 사례 게임입니다. 플랫폼이 실제로 글로벌 모바일 게임을 출시·운영 가능한 수준인지를 라이브 데이터로 입증합니다.
  - 당사가 보유한 닌자 IP를 활용하여 신규 유입 비용을 절감하고, 협약기간 내 베타 출시가 가능한 일정 안에 둡니다.

  ○ 게임 컨셉
  - 서바이벌(뱀서류 액션) + 하우징(영지 건설) + 4X(연맹 점령전)의 3단계 융합 모델입니다.
  - 2025년 실증 테스트(서바이벌 + 채집)에서 리텐션·수익 각 5%p 상승을 확인한 데이터를 기반으로 설계되었습니다.
  - 협약 종료 시점에 글로벌 베타 출시, '27.01 정식 출시 예정입니다.

### □ 자사 보유 역량 기반의 차별성과 경쟁력

  ○ 자체 엔진 + AI 워크플로우 + 라이브 운영의 수직 통합
  - 단편 콘텐츠 생성 AI 도구(Scenario, Layer 등)는 게임 엔진과 분리되어 있으며, 통합 게임 엔진(Unity, Unreal 등)의 AI 기능은 라이브 운영 자동화까지는 결합되어 있지 않습니다.
  - IDLE Forge는 자체 엔진과 AI 워크플로우, 라이브 운영 파이프라인을 한 코드베이스에 결합한 플랫폼으로, 시장에 명확한 경쟁자가 확인되지 않는 카테고리에 진입합니다.

  ○ 사내 라이브 게임으로 플랫폼을 실시간 검증
  - 외부 게임을 빌려 도구를 검증해야 하는 일반 AI 게임 도구 개발사와 달리, 당사는 자체 라이브 게임 2종을 보유하고 있어 플랫폼 개선과 실증을 매주 단위로 진행할 수 있습니다.

  ○ 검증 팀 그대로 이식
  - idlez 엔진과 AI 워크플로우를 다년간 함께 개발·운영해 온 동일 팀이 그대로 합류하여 본 사업을 추진합니다. 신규 법인이지만 핵심 인력의 협업 경험은 검증된 상태입니다.

### □ 기술 보호 계획

  ○ idlez 엔진의 핵심 로직(비동기 통신, 핫 패치 시스템 등)은 영업비밀로 보호하며, 사내 접근 권한을 단계적으로 관리합니다.

  ○ AI 콘텐츠 생성 워크플로우의 도메인 특화 스킬과 검증 에이전트 로직은 통합 자산으로 보호합니다. 2028년 외부 SaaS 공급 시 일부 인터페이스는 공개하되, 검증 로직과 파인튜닝된 게임 도메인 모델은 유료 라이선스 형태로 제공합니다.

  ○ 닌자 IP 관련 캐릭터·세계관 자산은 당사 소유로 관리하며, IDLE Forge로 생성된 2차 창작물도 동일 권리 체계를 적용합니다.
