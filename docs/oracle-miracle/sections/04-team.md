# 4. 기업 구성 (Team)

## 4-1. 기업구성 및 보유 역량

### 4-1-1. 대표자 및 핵심 인력

대표는 글로벌 라이브 게임 디렉팅과 AI 도구 활용·플랫폼 설계 양쪽에 필요한 역량을 보유합니다. 800만 다운로드 글로벌 방치형 게임의 초기 개발부터 출시까지 기획을 총괄했으며, 자체 운영 라이브 게임 2종(누적 170만 DL)을 다년간 운영하며 지표 분석·마케팅·사업 전반 노하우를 축적했습니다. AI를 활용한 웹·툴·시뮬레이션·프로토타입을 직접 개발할 수 있어 IDLE Forge 콘텐츠 생성 워크플로우 운영 방향을 직접 설계해 왔습니다.

## 4-2. ESG 경영 실천 계획

  ○ **환경(E) — OCI 클라우드 전환을 통한 탄소 발자국 최소화**
  자체 데이터센터를 운영하지 않고 OCI 클라우드 인프라를 활용하여 자체 서버 운영 대비 전력·냉각 자원을 절감합니다. 사용량 기반 자동 스케일로 유휴 자원을 두지 않으며, Oracle 글로벌 리전의 재생에너지 전환 추진과 정합한 친환경 운영 구조를 갖춥니다.

  ○ **사회(S) — 청년 일자리 창출과 국내 인디 게임 생태계 기여**
  협약기간 내 시니어 2 + 미들 3 등 5명을 추가 채용하고 '27년까지 9~11명 규모로 확장하며, 게임 기획·플랫폼·클라이언트·콘텐츠·데이터 운영 등 청년 친화 직무로 일자리를 확대합니다. '28년부터 IDLE Forge B2B 공급으로 국내 인디 게임사의 콘텐츠·운영 인건비 부담을 낮추고 글로벌 진출 기회를 확대하여 국내 인디 게임 생태계 활성화에 직접 기여합니다.

  ○ **지배구조(G) — 데이터·AI 거버넌스와 투명 경영**
  유저 개인정보는 OCI 보안 정책(암호화·접근 제어·감사 로그)과 리전 분리 운용을 기반으로 관리하며, 한국 PIPA·EU GDPR·중국 PIPL 등 지역별 개인정보 보호 법령을 동시 준수합니다. AI 콘텐츠 생성 단계에는 저작권·유사성 자동 검증과 활용 내역 사내 로그·감사를 적용하고, 정부지원사업비는 별도 계좌로 분리하여 분기 집행 보고 + '27년 매출 규모 도달 시 외부 회계법인 정기 검토를 도입합니다.
