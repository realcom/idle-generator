# ESG 경영 실천 계획 (4-2 항목용)

> 사업계획서 4-2 ESG 섹션 작성 골자. 인디 게임 스튜디오 + 클라우드 활용에 맞춰 적정 톤.

## 4-2-1. 친환경 경영 (Environmental)

### 핵심 메시지
**자체 데이터센터 없이 글로벌 라이브 게임 운영 → 탄소 배출 최소화**.

### 실천 항목

- **클라우드 네이티브 인프라**
  - OCI Compute / OKE 활용 → 자체 서버 운영 대비 전력·냉각 자원 90%+ 절감
  - 사용량 기반 자동 스케일 → 유휴 자원 제로
- **OCI 친환경 전력 정책 활용**
  - Oracle은 2025년까지 글로벌 클라우드 리전 100% 재생에너지 전환 공약 — 본 프로젝트가 이 인프라를 직접 활용
- **종이 제로 사무**
  - 전 업무 디지털화 (Notion, Slack 등), 인쇄 자료 최소화
- **개발 효율 = 환경 효율**
  - IDLE Forge로 콘텐츠 생산 효율 5x+ → 동일 산출물 대비 컴퓨팅·전력 사용량 감소

## 4-2-2. 사회적 책임 경영 (Social)

### 핵심 메시지
**누구나 즐길 수 있는 게임 · 청년 일자리 창출 · 게임 산업 AI 전환 기여**.

### 실천 항목

- **인디 게임사의 AI 네이티브 전환 모델 제시**
  - IDLE Forge 노하우를 향후 컨퍼런스·블로그를 통해 한국 게임 업계와 공유
  - '28년 SaaS 공급 시점에는 인디 게임사 N개사 직접 지원
- **다양성과 접근성**
  - 다국어 자동 로컬라이징(OCI Language)으로 비영어권 유저 접근성 보장
  - 캐릭터·콘텐츠 설계에서 다양한 문화권·성별 표현 (AI 콘텐츠 검수 시 다양성 가이드라인 적용)
- **청년 일자리 창출**
  - 협약기간 내 4명 추가 채용 (시니어 2 + 미들 2), '27년까지 8~10명 규모로 확장
  - 게임 개발·기획·아트·AI 워크플로우 등 청년 친화 직무
- **건전한 게임 운영**
  - 청소년 보호 가이드라인 준수, 결제 한도·이용시간 자율 안내
  - 부정 결제·치팅 자동 탐지 (OCI Logging Analytics)로 공정한 게임 환경 보장

## 4-2-3. 투명 경영 (Governance)

### 핵심 메시지
**소규모 법인이지만 투명한 의사결정 구조와 윤리 경영 기반 확립**.

### 실천 항목

- **명확한 지분·권한 구조**
  - 대표 100% 지분 보유로 의사결정 명확화
  - 합류 인력에 대한 스톡옵션·인센티브 제도 단계적 도입 (성과 공유)
- **데이터 거버넌스**
  - 유저 개인정보 OCI 보안 정책 (암호화·RBAC·Audit Logs) 기반 관리
  - 한국 개인정보보호법·EU GDPR·중국 PIPL 준수 자동화 (OCI 리전 분리 운용)
- **AI 윤리**
  - AI 콘텐츠 생성 시 저작권·유사성 자동 검증 (content-reviewer)
  - AI 활용 내역 사내 로그로 기록·감사 가능 (OCI Audit Logs)
- **윤리경영 가이드라인**
  - 상호 존중 조직 문화, 정기 1:1 피드백 체계
  - 외주·협력사 대금 정시 지급, 불공정 거래 금지
- **재무 투명성**
  - 정부지원사업비 별도 계좌 관리, 분기 단위 집행 보고
  - 외부 회계 검토 ('27년 매출 규모 도달 시 회계법인 위탁)
