# Harness Examples

이 디렉터리는 확인·데모용으로 보존하는 생성 산출물이다. 활성 엔진 리소스인 `PatchResources`로 취급하지 않는다.

- `patchresources/`: Unity `PatchResources` 전체 예시. JSON은 최신 `harness/content/idlez` 컴파일 산출물로 갱신한다.

실제 Unity 기본값은 `engine/client/Client/Assets/PatchResources`에 둔다. 예시 번들은 end-to-end 로드 경로를 검증할 때만 의도적으로 복사한다.
