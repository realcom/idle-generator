# Ninja2 Flat Arcade Survivors Scene E

## Intent

사용자 레퍼런스처럼 더 플랫하고 읽기 쉬운 모바일 서바이벌 전장을 시험한 후보. `scene_d`보다 반복 적, 드롭, 피해 숫자, HUD 가시성이 좋아졌지만, 캐릭터와 적 렌더링이 아직 약간 풍부해서 최종 플랫 기준보다는 중간 후보로 둔다.

## Art Anchor

- 주인공은 갈색 머리, 크림색 망토, 붉은 스카프, 손등불로 식별한다.
- 적은 잎 가면 임프, 검은 숯 정령, 가시 덩어리, 버섯류로 구성한다.
- 전투 보상은 파란 EXP 조각, 노란 코인, 청록 영혼불, 목재 상자/묶음으로 구분한다.
- 성소 테마는 등불, 잎 부적, 영혼불로만 얹고 닌자성은 줄인다.

## Composition Rules

- Portrait 9:16 모바일 화면.
- 단색 잔디 필드 위에 반복 적과 드롭을 배치한다.
- 중앙에는 주인공과 원형 스킬 범위가 있고, 외곽에서 적이 계속 밀려온다.
- 배경 장식은 작고 드문드문 두며, 적/드롭과 경쟁하지 않는다.

## UI Direction

- 상단 타이머/스테이지 블록과 재화 카운터를 두껍고 단순하게 유지한다.
- 좌측에는 작은 스킬/아이템 슬롯을 세로로 둔다.
- 하단 좌측 조이스틱, 하단 우측 업그레이드/초상 버튼 구조를 유지한다.
- UI는 레퍼런스처럼 기능 우선, 설명 텍스트 없이 아이콘 중심으로 둔다.

## Implementation Notes

- 분위기는 좋지만 최종 플랫 기준은 `ninja2_flat_arcade_survivors_scene_f`를 우선한다.
- 이 방향을 쓰려면 캐릭터/적의 그라데이션과 세부 음영을 30% 더 줄인다.
- 적 스프라이트는 3~4종만 반복해도 충분히 풍성해 보인다.
