# Ninja2 Cozy Dark Character Style A

## Intent

닌자 중심 컨셉에서 벗어나, `귀엽지만 어딘가 불길한 작은 성소 공동체` 방향의 캐릭터 스타일을 탐색한다. UI보다 캐릭터와 주민/추종자 원형을 먼저 정리해 하우징과 출격 루프의 감정적 중심을 만든다.

## Art Anchor

- 주인공은 닌자/암살자가 아니라 등불을 들고 성소를 이끄는 작은 수호자다.
- 핵심 실루엣은 후드 망토, 큰 눈, 등불, 의식 펜던트, 작은 가방이다.
- 주민은 동물/숲정령 계열로 역할이 바로 읽혀야 한다: 버섯 정원사, 촛불 시종, 광부, 목수, 요리사.
- 양, 왕관, 직접적인 컬트 상징은 금지한다. 위험한 복제가 아니라 별도의 성소 공동체로 보여야 한다.

## Character Rules

- Cute first, spooky second.
- 주인공은 호감형이지만 너무 아기 같지 않아야 한다.
- 주민은 작은 몸, 큰 눈, 직업 도구, 한 가지 명확한 색 포인트를 가진다.
- 액세서리는 등불, 부적, 작은 종, 씨앗 주머니, 나뭇잎 장식 같은 생활형 의식 도구를 사용한다.
- 공포/고어 대신 따뜻한 불빛과 수상한 의식 분위기로 긴장을 만든다.

## Palette

- Warm cream: 후드/천/양피지 계열
- Moss green: 숲/주민/농장
- Charcoal brown: 외곽선/그림자/나무
- Muted red: 문장/리본/작은 의식 포인트
- Soft gold: 등불/축복/CTA
- Pale teal: 영혼불/마법 자원

## Implementation Notes

- 이 스타일은 기존 `ninja2_hero_character_concept_a`보다 닌자성이 약하고, 하우징/주민 관리 루프에 더 잘 붙는다.
- 추후 캐릭터 에셋 생성 시 주인공 1종 + 주민 직업 5종을 먼저 만들고, 전투 적은 배경 숲의 변질된 정령으로 분리한다.
- UI 초상에는 주인공의 등불과 후드 실루엣이 반드시 보여야 한다.
- 주민은 하우징 건물과 1:1로 연결할 수 있다: 정원사-허브밭, 광부-채석장, 목수-작업장, 시종-성소, 요리사-식당.
