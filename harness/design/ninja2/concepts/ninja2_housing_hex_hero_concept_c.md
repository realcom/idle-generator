# Ninja2 Housing Hex Hero Concept C

## Intent

`ninja2` 하우징 홈 화면의 캐릭터 매력 보강 후보. Concept B의 캐주얼 헥스 영지 방향은 유지하되, 주인공이 너무 평범하고 매력이 약한 문제를 해결하기 위해 상단 초상과 중앙 아바타를 모두 강한 정체성 앵커로 세웠다.

## Art Anchor

- 주인공은 복면 닌자가 아니라 수집형 스킨을 붙이고 싶은 캐주얼 히어로다.
- 핵심 실루엣은 둥근 얼굴, 또렷한 눈, 살짝 헝클어진 머리, 붉은 스카프, 작은 머리띠/문장 포인트다.
- 닌자성은 검은 가면보다 도장, 깃발, 수련 도구, 민첩한 자세로 표현한다.
- 건물보다 캐릭터의 얼굴 완성도가 높아야 하고, 상단 프로필 초상만 봐도 게임의 주인공이 기억되어야 한다.

## Composition Rules

- Portrait 9:16 모바일 화면 기준.
- 상단 좌측 프로필 패널에 매력적인 캐릭터 초상을 크게 둔다.
- 중앙 도장 앞에는 같은 주인공의 전신 아바타를 세워 화면의 감정 중심을 만든다.
- 헥스 보드는 Concept B와 동일하게 영지 확장/건설/업그레이드가 읽히는 구조를 유지한다.
- 선택 건물 정보 패널과 `탐험 출발` CTA는 하단에 두되, 캐릭터를 가리지 않는다.

## UI Direction

- Profile: 얼굴, 레벨, 진행 바, 전투력 정도만 간결하게 보여준다.
- Hero: 검은 복면 금지. 밝은 피부톤, 빨간 스카프, 작은 문장으로 브랜드 컬러를 만든다.
- Buildings: 도장, 대나무 작업장, 창고, 수련마당, 허브밭, 채석장 등 생활형 건물 중심.
- CTA: `탐험 출발`은 깃발/에너지 아이콘 중심. 큰 칼 아이콘은 피한다.
- Progress: 일일 임무, 잠금 타일 비용, 업그레이드 가능 상태를 초록/골드 액센트로 명확히 구분한다.

## Implementation Notes

- 현재 캐릭터 방향의 우선 후보. UI recipe나 캐릭터 에셋 생성 시 이 초상/전신의 비율과 컬러 포인트를 기준으로 삼는다.
- 생성 이미지의 이름/텍스트는 placeholder다. 실제 주인공명은 별도 네이밍 단계에서 확정한다.
- 캐릭터는 작아져도 얼굴과 빨간 스카프가 남아야 하므로, 전신 스프라이트 제작 시 머리/스카프 비율을 크게 유지한다.
- Concept B는 보드 톤 레퍼런스로 유지하고, Concept C는 캐릭터 매력/주인공성 레퍼런스로 우선한다.
- 추후 Survival 전투 화면에서도 같은 캐릭터가 더 역동적인 포즈로 이어져야 한다.
