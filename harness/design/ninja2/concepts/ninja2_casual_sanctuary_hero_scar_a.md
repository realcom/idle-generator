# Ninja2 Casual Sanctuary Hero Scar A

## Intent

`casual_sanctuary` 피벗의 메인 캐릭터에 이마 흉터를 추가한 후보. 다정하지만 결의 있는 보호자 느낌을 강화하면서도, 특정 기존 캐릭터를 직접 복제하지 않고 성소 수호자 고유의 시그니처로 만든다.

## Art Anchor

- 핵심 앵커는 따뜻한 갈색 머리, 크림색 후드 망토, 모스그린 안감, 붉은 스카프, 원형 펜던트, 등불, 이마 흉터다.
- 이마 흉터는 잎/불꽃/가지 문양처럼 작고 상징적인 형태로 둔다.
- 흉터는 캐릭터의 개인 서사 표시이며, 고어/상처 강조가 아니다.
- 직접 식별 요소는 피한다: 체크무늬 외투, 귀걸이, 카타나 중심 실루엣, 특정 애니메이션 문양.

## Character Rules

- Cute 70 / Mystery 30.
- 얼굴은 밝고 호감형이지만, serious bust에서는 보호자다운 결의를 보여준다.
- 흉터는 초상 크롭에서도 읽혀야 하며, 스킨을 바꿔도 유지되는 영구 식별자다.
- 전투/탐험 포즈는 큰 무기보다 등불, 작은 도구, 민첩한 움직임으로 표현한다.
- 빨간 스카프와 이마 흉터가 주인공의 브랜드 컬러/문장이 된다.

## Implementation Notes

- 현재 캐주얼 성소 피벗의 메인 캐릭터 우선 후보로 올린다.
- 이전 `ninja2_casual_sanctuary_hero_a`보다 보호자/서사성이 강하다.
- UI 초상용 crop은 우측 상단 serious bust 또는 우측 중단 smiling bust를 기준으로 테스트한다.
- 인게임 작은 스프라이트에서는 이마 흉터가 사라질 수 있으므로, 빨간 스카프와 등불까지 함께 식별자로 유지한다.
