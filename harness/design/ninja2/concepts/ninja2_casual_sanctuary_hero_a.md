# Ninja2 Casual Sanctuary Hero A

## Intent

모바일 캐주얼 감각을 유지하면서 `작은 성소 수호자` 계열의 메인 캐릭터를 새로 잡은 후보. 닌자/복면/큰 무기에서 벗어나, 홈 화면과 스킨 수집에 어울리는 밝고 호감형 주인공을 목표로 한다.

## Art Anchor

- 핵심 실루엣은 부드러운 후드 망토, 풍성한 갈색 머리, 따뜻한 눈, 작은 등불, 원형 펜던트다.
- 캐릭터는 성소 관리자이자 탐험가이며, 공포/컬트 리더가 아니다.
- 붉은색은 작은 스카프/리본 포인트로만 쓰고, 전체 톤은 크림/모스그린/골드 중심으로 둔다.
- 작은 유령 부적, 잎 장식, 등불, 씨앗 주머니는 이 세계의 마스코트 아이템으로 확장 가능하다.

## Character Rules

- Cute 70 / Mystery 30.
- 얼굴은 밝고 친근해야 하며, 표정은 미소/호기심/가벼운 자신감이 중심이다.
- 후드와 망토는 신비감을 주되, 검거나 무겁게 만들지 않는다.
- 전투 포즈에서도 큰 무기보다 등불, 작은 도구, 민첩한 움직임을 사용한다.
- 초상/전신/작은 아이콘으로 줄여도 머리 실루엣과 등불이 남아야 한다.

## Implementation Notes

- 현재 캐주얼 성소 피벗의 메인 캐릭터 우선 후보.
- 이전 `ninja2_hero_character_concept_a`보다 닌자성이 약하고, `cozy_dark_character_style_a`보다 모바일 캐주얼 톤이 밝다.
- UI 초상은 웃는 bust를 기준으로 삼고, 홈 화면 전신 아바타는 등불을 든 기본 포즈를 기준으로 삼는다.
- 스킨 확장은 망토 패턴, 등불 모양, 펜던트 문장, 머리 장식으로 한다.
