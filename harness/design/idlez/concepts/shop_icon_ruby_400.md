# Shop Icon Ruby 400

## Intent

루비 소량 상품을 한눈에 읽히는 기본 프리미엄 통화 아이콘으로 정의한다.

## Art Anchor

- `hamster_shop_modal_concept_a`의 붉은 루비, 목재/양피지 상점 톤을 따른다.
- 두꺼운 다크 브라운 외곽선과 장난감 같은 광택을 유지한다.

## Composition Rules

- 숫자와 텍스트를 이미지에 굽지 않는다.
- 작은 루비 더미만 중앙에 두어 64px 카드에서도 즉시 읽히게 한다.
- 카드 배경, 버튼, 가격은 런타임 UI가 담당한다.

## UI Direction

- 루비 400 카드의 낮은 가격대와 기본 패키지 느낌을 위해 가장 단순한 보석 더미를 사용한다.
