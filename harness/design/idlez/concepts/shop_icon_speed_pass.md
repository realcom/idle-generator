# Shop Icon Speed Pass

## Intent

배속권을 빠른 전투 속도 권한으로 읽히는 스톱워치 아이콘으로 정의한다.

## Art Anchor

- 상점 카드의 골드/브라운 계열과 빠른 움직임을 나타내는 흰색 속도선을 조합한다.
- 중앙에는 이중 재생 삼각형을 넣어 배속 기능을 전달한다.

## Composition Rules

- `2x` 같은 텍스트는 아이콘에 넣지 않고 카드 설명에서 처리한다.
- 스톱워치 외곽과 빠른 움직임 선이 64px에서도 유지되어야 한다.
- 투명 정사각형 에셋으로 런타임 카드 위에 배치한다.

## UI Direction

- 광고 제거권보다 더 동적인 아이콘으로, 기능성 상품 간 시각적 구분을 만든다.
