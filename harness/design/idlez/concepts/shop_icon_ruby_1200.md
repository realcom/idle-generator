# Shop Icon Ruby 1200

## Intent

중간 가격대 루비 상품을 보석 주머니 실루엣으로 분리한다.

## Art Anchor

- `hamster_shop_modal_concept_a`의 루비 주머니 이미지를 기준으로 한다.
- 브라운 가죽 주머니, 붉은 루비, 골드 하이라이트를 같은 재질감으로 맞춘다.

## Composition Rules

- 보너스 배지는 런타임 카드가 별도로 그리므로 아이콘에는 넣지 않는다.
- 주머니와 흘러나온 루비를 중앙에 배치한다.
- 텍스트, 숫자, 가격은 이미지에 포함하지 않는다.

## UI Direction

- 루비 400보다 풍성하지만 루비 3000보다 작게 읽히는 중간 단계 상품 아이콘이다.
