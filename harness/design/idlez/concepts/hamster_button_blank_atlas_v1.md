# Hamster Button Blank Atlas V1

## Intent

`hamster_button_skin_sheet_v1`을 실제 Phaser/CSS에 적용하기 위한 텍스트 없는 버튼 몸체 atlas. 버튼 재질은 PNG asset으로 사용하고, 텍스트/숫자/아이콘은 HTML 레이어에서 얹는다.
