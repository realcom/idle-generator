# Shop Icon Ruby 3000

## Intent

최고 가격대 루비 상품을 보물상자 실루엣으로 강조한다.

## Art Anchor

- `hamster_shop_modal_concept_a`의 보물상자 루비 상품을 따른다.
- 골드 장식, 두꺼운 외곽선, 붉은 루비 광택을 기존 상점 스킨과 통일한다.

## Composition Rules

- 상자는 열려 있고 루비가 넘쳐야 한다.
- 보너스 배지와 가격은 런타임 카드 레이어에서 처리한다.
- 작은 크기에서도 상자와 루비가 분리되어 보여야 한다.

## UI Direction

- 루비 라인업의 최상위 상품으로 가장 부피감 있는 아이콘을 사용한다.
